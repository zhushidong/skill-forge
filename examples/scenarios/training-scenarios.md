# 培训场景样例

## 场景 11：新人话术演练

### 原始案例

```markdown
# 新人话术演练案例

## 背景
新人：小李，入职1周
产品：999元课程
场景：价格异议演练

## 演练记录

导师：小李，我们来演练一下价格异议处理。我扮演客户，你扮演销售。准备好了吗？

小李：准备好了。

导师：好的，开始。

导师（客户）：你们这个课程太贵了，999元，我考虑一下。

小李：嗯...这个课程是999元，但是...内容很丰富。

导师（客户）：内容丰富有什么用，太贵了。

小李：那...您可以分期付款。

导师（客户）：分期也要付那么多钱。

小李：那...我帮您问一下能不能便宜点。

导师（客户）：好的，你去问吧。

（演练结束）

导师：小李，你刚才的处理有几个问题：

1. 没有先认同客户的顾虑
2. 没有了解客户的真实预算
3. 直接说分期，没有展示价值
4. 说要问能不能便宜，这是直接降价

小李：明白了，那应该怎么处理呢？

导师：首先，要先认同客户的顾虑。比如："我理解您的顾虑，价格确实是一个重要考虑因素。"

小李：然后呢？

导师：然后，要了解客户的真实预算。比如："您理想的预算是多少？我可以帮您看看有没有合适的方案。"

小李：明白了。

导师：接着，要展示价值而非降价。比如："这个方案可以帮助您平均下来每天只需要3.3元。"

小李：这个话术不错。

导师：最后，可以提供分期选项，但不是直接降价。比如："我们还有分期付款方案，可以减轻一次性支付的压力。"

小李：明白了，我再试一次。

## 结果
新人理解话术要点，需要继续练习。
```

## 场景 12：管理者点评

### 原始案例

```markdown
# 管理者点评案例

## 背景
管理者：王经理
场景：团队周会复盘

## 点评记录

王经理：好的，我们来复盘一下上周的销售情况。

王经理：小李，你上周处理了10个客户，成交了2个，成功率20%。你觉得自己哪里做得好，哪里做得不好？

小李：我觉得我话术还不够熟练，有时候不知道怎么回应客户。

王经理：具体是哪些场景？

小李：主要是价格异议，客户一说太贵了，我就不知道怎么处理。

王经理：明白了。那你有没有用我们的话术模板？

小李：有，但是感觉效果不好。

王经理：那你有没有分析过为什么效果不好？

小李：没有。

王经理：那这样，你把上周的价格异议案例整理一下，我们下周一起分析。

小李：好的。

王经理：小张，你上周处理了15个客户，成交了5个，成功率33%。你做得很好，有什么经验可以分享吗？

小张：我觉得关键是先了解客户的需求，然后针对性地推荐。

王经理：具体怎么做？

小张：我会先问客户是自己用还是公司用，预算多少，然后根据这些信息推荐合适的方案。

王经理：很好，这个经验值得分享。下周让大家一起学习一下。

## 结果
管理者了解团队情况，制定改进计划。
```

## 场景 13：优秀案例提炼

### 原始案例

```markdown
# 优秀案例提炼案例

## 背景
销售：小张
产品：999元课程
场景：成功成交案例

## 案例记录

小张：客户一开始说太贵了，我没有直接降价，而是先认同他的顾虑，然后了解他的预算。

小张：他说预算只有500元，我推荐了基础版，只要499元。

小张：但是基础版没有答疑和咨询，他有点犹豫。

小张：我告诉他，基础版包含核心内容，而且如果后续需要答疑，可以单独购买。

小张：最后他买了基础版，还说以后有需要会升级。

## 提炼过程

导师：小张，你这个案例处理得很好。我们来提炼一下，看看有哪些经验可以复制。

小张：好的。

导师：首先，你没有直接降价，而是先认同客户的顾虑。这是对的。

小张：是的，我觉得先认同客户，客户会更愿意听你说。

导师：其次，你了解了客户的真实预算，然后推荐了合适的方案。

小张：是的，如果我不知道他的预算，可能会推荐更贵的方案，他就不会买了。

导师：最后，你提供了升级路径，让客户觉得有后路。

小张：是的，这样客户不会觉得被套牢了。

导师：很好，我们把这个案例整理成一个Skill，可以让其他销售学习。

## 结果
提炼出优秀案例，可以复制。
```

## 场景 14：失败案例复盘

### 原始案例

```markdown
# 失败案例复盘案例

## 背景
销售：小李
产品：999元课程
场景：客户流失案例

## 案例记录

小李：客户说太贵了，我直接给他打了8折。

小李：客户还是觉得贵，我又打了7折。

小李：最后客户还是没有买。

## 复盘过程

导师：小李，我们来复盘一下这个案例。你觉得哪里做得不好？

小李：我觉得我不应该直接降价。

导师：为什么？

小李：因为降了两次价，客户还是觉得贵，说明问题不在价格。

导师：很好，那问题在哪里？

小李：可能是我没有了解客户的真实需求。

导师：对，你直接降价，没有先了解客户为什么觉得贵。是预算不够？还是觉得不值？还是在比较竞品？

小李：我没有问。

导师：那应该怎么处理？

小李：应该先问客户为什么觉得贵，然后针对性地处理。

导师：很好，我们把这个案例整理成一个教训，以后避免再犯。

## 结果
新人认识到错误，知道如何改进。
```
