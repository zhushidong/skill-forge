# 销售场景样例

## 场景 1：客户嫌贵

### 原始案例

```markdown
# 客户嫌贵案例

## 背景
客户：张总，中小企业老板
产品：999元线上课程
来源：小红书广告

## 对话记录

销售：张总您好，看了我们的课程介绍，感觉怎么样？

客户：内容还行，就是价格有点贵。

销售：这个课程是999元，包含30节视频课+1年社群答疑+3次1v1咨询。

客户：我知道，但是999对我们小公司来说还是有点压力。人家XXX的课才299。

销售：张总，您不能这么比。299的课只有视频，没有答疑和咨询。我们这个是全套服务。

客户：但是299的也能学啊，反正都是那些内容。

销售：张总，这个不一样的。我们的课程是实战派，学完就能用。299那个是理论派，学了还是不会用。

客户：你说的有道理，但是我还是觉得贵。

销售：那这样，我给您申请一个优惠，899行不行？

客户：899...还是有点贵。

销售：那我再给您减100，799，这已经是最低价了。

客户：我再考虑考虑吧。

## 结果
客户流失，未成交。
```

      action: 了解客户的真实预算
      script: "您理想的预算是多少？我可以帮您看看有没有合适的方案。"
    - step: 3
      action: 展示价值而非降价
      script: "这个方案可以帮助您平均下来每天只需要3.3元。"

      action: 提供分期/套餐选项
      script: "我们还有分期付款方案，可以减轻一次性支付的压力。"

  - 争辩客户不懂价值
  - 过早推套餐
---

# 价格异议处理-价值重构法

## 解决的问题
客户以"太贵了"为由拒绝购买时，如何有效回应。

## 信号识别
当客户说出以下关键词时，激活此 Skill：
- "太贵了"
- "预算不够"
- "别人家便宜"

## 执行步骤

### 步骤 1：认同顾虑
"我理解您的顾虑，价格确实是一个重要考虑因素。"

### 步骤 2：了解真实预算
"您理想的预算是多少？我可以帮您看看有没有合适的方案。"

### 步骤 3：展示价值
"这个方案可以帮助您平均下来每天只需要3.3元。"

### 步骤 4：提供选项
"我们还有分期付款方案，可以减轻一次性支付的压力。"

## 禁用行为
- 直接降价
- 争辩客户不懂价值
- 过早推套餐

## 场景 2：客户说再考虑

### 原始案例

```markdown
# 客户说再考虑案例

## 背景
客户：李经理，中型企业采购
产品：5000元企业培训服务
来源：展会

## 对话记录

销售：李经理，上次展会聊的培训服务，您考虑得怎么样了？

客户：嗯，我看了你们的方案，挺好的。但是我需要再考虑考虑。

销售：李经理，您主要是哪方面需要考虑呢？

客户：就是...我需要和领导汇报一下，看看领导的意见。

销售：理解，那您大概什么时候能和领导沟通完？

客户：这个不确定，领导最近比较忙。

销售：那这样，我先给您发一份详细的方案书，方便您向领导汇报。您看可以吗？

客户：好的，你发吧。

销售：好的，我今天就发。另外，您觉得领导可能会关心哪些方面？

客户：领导主要关心效果和成本。

销售：明白了，我会在方案书里重点突出这两点。

## 结果
客户承诺汇报，但没有明确时间。
```

```markdown
# 客户拿竞品压价案例

客户：这个数据靠谱吗？

销售：我们有学员案例可以分享给您看。您现在方便看吗？

客户：好的，你发给我看看。

## 结果
客户要求看案例，未成交。
```

## 场景 4：客户没有预算

### 原始案例

```markdown
# 客户没有预算案例

## 背景
客户：赵经理，创业公司
产品：2000元运营课程
来源：知乎广告

## 对话记录

销售：理解，创业初期确实预算紧张。那您觉得这个课程对您有帮助吗？

销售：那这样，我们有一个分期付款方案，可以分3期，每期只要700元。您觉得怎么样？

客户：分期...我还是觉得贵。

客户：最多500元。

销售：500元...我帮您看看有没有其他方案。您主要是想学哪方面？

销售：那这样，我们有一个基础版，只要499元，包含社群运营的核心内容。您看可以吗？

客户：基础版有什么区别？

销售：基础版只有视频，没有答疑和1对1指导。但是核心内容都在。

客户：那可以，我考虑一下。

## 结果
客户表示考虑，未成交。
```

## 场景 5：客户要找领导审批

### 原始案例

```markdown
# 客户要找领导审批案例

## 背景
客户：刘主管，大型企业
产品：10000元企业培训
来源：LinkedIn

## 对话记录

销售：刘主管，上次聊的培训方案，您考虑得怎么样了？

客户：方案挺好的，但是我需要找领导审批。

销售：理解，那您觉得领导可能会关心哪些方面？

客户：领导主要关心ROI和实施风险。

销售：明白了，我准备一份ROI分析报告，方便您向领导汇报。您看可以吗？

客户：好的，你准备吧。

销售：另外，您觉得领导大概什么时候能审批？

客户：这个不确定，领导最近在出差。

销售：那这样，我先把材料发给您，等领导回来您再汇报。您看可以吗？

客户：好的。

销售：好的，我今天就发。另外，您需要我一起参加汇报吗？

客户：不用了，我自己汇报就行。

销售：好的，那您汇报完告诉我结果，我随时配合。

## 结果
客户承诺汇报，未成交。
```
